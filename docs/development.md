# Разработка

Используется:
- angular:^20
- taiga-ui:^4
- firebase

## Сервер разработки

Необходимо поднять эмулятор firebase:
```shell
npm run firebase
```
и запустить сервер разработки angular
```shell
ng serve
```

## Firebase

Настройки деплоя и эмулятора хранится в файле [firebase.json](..%2Ffirebase.json)
