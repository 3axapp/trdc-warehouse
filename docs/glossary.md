# Глоссарий

**1. Производство**
Процесс создания готовой продукции из материалов с использованием определённых технологических операций.

**2. Рецепт**
Запись, содержащая описание производимой позиции и перечень необходимых для её производства материалов с указанием их количества. Рецепт является основой для планирования и осуществления производственного процесса.

**3. Позиция**
Уникальный идентифицируемый объект в системе (по полю `code`), который может быть:
* **Материал:** Используется в производстве.
* **Чип:** Полупродукт, создаваемый на одном из этапов производства.
* **Упаковка:** Готовый продукт, упакованный для отправки.
* **Отгрузка:** Партия готовой продукции, подготовленная к отправке клиенту.

**4. Материал**
Базовая позиция, используемая на начальном этапе производства согласно рецепту. Поставляется на склад через карантин.

**5. Полупродукт, Готовый продукт**
Позиция, полученная в результате одного из внутренних этапов производства. Может являться материалом для последующих этапов (например, для создания упаковки, отгрузки).

**6. Отгрузка**
Партия или группа упаковок (готовой продукции), сформированная для отправки конкретному клиенту или по конкретному заказу.

**7. Склад**
Логистическая и учётная единица системы, предназначенная для хранения и учёта позиций: материалов, полупродуктов, готовых продуктов.

**8. Поставка**
Запись на складе, фиксирующая наличие позиции. Создаётся в результате контроля качества из карантина или как результат производства.

**9. Карантин**
Зона входного приёма материалов. Материалы поступают через карантинные счета и проходят контроль качества перед попаданием на склад.

**10. Карантинный счёт**
Документ, фиксирующий поступление партии материалов от поставщика. Содержит дату, номер, уникальный лот, поставщика и список позиций с количеством.

**11. Контроль качества (Входной контроль)**
Процедура проверки качества позиций из карантинного счёта. Проводится для проверяемых позиций. В результате создаётся поставка на складе с уникальным лотом.

**12. Тип позиции "Проверяемый"**
Атрибут позиции, который указывает на обязательность прохождения процедуры контроля качества перед допуском в производство.

**13. Резерв**
Предварительное бронирование материалов со склада для производства чипов. Зарезервированные материалы списываются со склада и могут быть использованы для производства или возвращены обратно.

**14. Исполнитель (Пользователь)**
Зарегистрированный пользователь системы. Каждому пользователю присваивается уникальный последовательный номер, который используется при формировании производственных лотов.

**15. Лот**
Уникальный идентификатор партии. Для карантинного КК формат: `КОД-ДДММГГ-НОМЕР_СЧЁТА-N`. Для производства чипов: `ДДММГГ-НОМЕР_ПОЛЬЗОВАТЕЛЯ-N`.

**16. Путь производства (Технологическая цепочка, производственный маршрут)**
Последовательность этапов преобразования материалов в готовую продукцию.
В описанной системе включает этапы:  
**Карантин → Склад → Резерв → Чип → Упаковка → Отгрузка**.  
Каждый переход между этапами регламентируется соответствующим рецептом или процедурой.
