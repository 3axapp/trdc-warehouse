# Глоссарий

**1. Производство**  
Процесс создания готовой продукции из материалов с использованием определённых технологических операций.

**2. Рецепт**  
Запись, содержащая описание производимой позиции и перечень необходимых для её производства материалов с указанием их количества. Рецепт является основой для планирования и осуществления производственного процесса.

**3. Позиция**  
Уникальный идентифицируемый объект в системе(по полю `code`), который может быть:
*   **Материал:** Используется в производстве.
*   **Чип:** Полупродукт, создаваемый на одном из этапов производства.
*   **Упаковка:** Готовый продукт, упакованный для отправки.
*   **Отгрузка:** Партия готовой продукции, подготовленная к отправке клиенту.

**4. Материал**  
Базовая позиция, используемая на начальном этапе производства согласно рецепту. Поставляется на склад извне.

**5. Полупродукт, Готовый продукт**  
Позиция, полученная в результате одного из внутренних этапов производства. Может являться материалом для последующих этапов (например, для создания упаковки, отгрузки).

**6. Отгрузка**  
Партия или группа упаковок (готовой продукции), сформированная для отправки конкретному клиенту или по конкретному заказу.

**7. Склад**  
Логистическая и учётная единица системы, предназначенная для хранения и учёта позиций: материалов, полупродуктов, готовых продуктов.

**8. Поставка**  
Процесс и документ, фиксирующий поступление материалов на склад от внешних поставщиков.

**9. Контроль качества (Входной контроль)**  
Процедура проверки качества позиций, поступивших на склад в рамках поставки. Проводится выборочно или в полном объёме в зависимости от типа позиции.

**11. Тип позиции "Проверяемый"**  
Атрибут позиции, который указывает на обязательность прохождения процедуры контроля качества после поставки на склад перед допуском в производство.

**12. Путь производства (Технологическая цепочка, производственный маршрут)**  
Последовательность этапов преобразования материалов в готовую продукцию. 
В описанной системе включает этапы: **Материалы -> Чип -> Упаковка -> Отгрузка**. Каждый переход между этапами регламентируется соответствующим рецептом.
